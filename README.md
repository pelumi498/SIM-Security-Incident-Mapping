# SIM - Security Incident Mapping

## SEN202 Software Engineering Project

### Description
SIM is a simple web-based application that allows students, staff, and landlords to report security incidents on and around campus.

### Features
- Report security incidents
- View reported incidents
- Anonymous reporting
- Admin management

### Technologies Used
- HTML
- CSS
- JavaScript
